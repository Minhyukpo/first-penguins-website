자주 묻는 질문

Goal-Illa 앱에 대한 궁금한 점들을 확인해보세요.

전체 다운로드 계정 기능 문제해결
Goal-Illa 앱을 어떻게 다운로드하나요?

Goal-Illa 앱은 App Store와 Google Play Store에서 다운로드할 수 있습니다.

  • iOS: App Store에서 "Goal-Illa"를 검색하여 다운로드
  • Android: Google Play Store에서 "Goal-Illa"를 검색하여 다운로드

앱은 무료로 다운로드할 수 있으며, 일부 프리미엄 기능은 인앱 구매를 통해 이용할 수 있습니다.

앱이 내 기기와 호환되나요?

Goal-Illa는 다음과 같은 요구사항을 만족하는 기기에서 사용할 수 있습니다:

  • iOS: iOS 12.0 이상
  • Android: Android 6.0 (API level 23) 이상

앱이 정상적으로 작동하지 않는다면 기기 사양을 확인해주세요.

계정을 잊어버렸어요. 어떻게 찾나요?

계정을 찾는 방법은 다음과 같습니다:

  1. 앱 로그인 화면에서 "계정 찾기" 버튼을 클릭
  2. 가입 시 사용한 이메일 주소를 입력
  3. 이메일로 전송된 링크를 클릭하여 비밀번호 재설정

이메일을 받지 못했다면 스팸 폴더를 확인해보세요.

계정을 삭제하고 싶어요

계정 삭제는 다음 단계를 따라 진행하세요:

  1. 앱 설정 > 계정 관리로 이동
  2. "계정 삭제" 옵션 선택
  3. 삭제 확인 절차 완료

주의: 계정 삭제 시 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다.

내 데이터는 안전한가요?

네, 사용자의 데이터 보안을 최우선으로 생각합니다:

  • 암호화: 모든 데이터는 AES-256 암호화로 보호됩니다
  • 백업: 정기적인 자동 백업으로 데이터 손실을 방지합니다
  • 개인정보: 개인정보보호법을 준수하여 데이터를 처리합니다
  • 서버 보안: 최신 보안 기술로 서버를 보호합니다
목표를 어떻게 설정하나요?

목표 설정은 매우 간단합니다:

  1. 앱 메인 화면에서 "+" 버튼 클릭
  2. 목표 제목과 설명 입력
  3. 목표 유형 선택 (습관, 프로젝트, 일회성 등)
  4. 목표 기간과 마일스톤 설정
  5. 알림 설정 (선택사항)

SMART 원칙에 따라 구체적이고 측정 가능한 목표를 설정하는 것을 권장합니다.

진행률은 어떻게 확인하나요?

진행률 확인 방법:

  • 대시보드: 메인 화면에서 전체 목표 진행률 확인
  • 개별 목표: 각 목표를 클릭하여 상세 진행률 확인
  • 통계: 주간/월간 통계로 성과 분석
  • 차트: 시각적 그래프로 진행 상황 파악
앱이 자주 업데이트되나요?

Goal-Illa는 지속적인 개선을 위해 정기적으로 업데이트됩니다:

  • 주요 업데이트: 월 1-2회 새로운 기능 추가
  • 버그 수정: 발견 즉시 수정 업데이트
  • 보안 업데이트: 보안 강화를 위한 업데이트

자동 업데이트를 활성화하면 최신 버전을 자동으로 받을 수 있습니다.

앱이 느리거나 멈춰요

앱 성능 문제 해결 방법:

  1. 앱 재시작: 앱을 완전히 종료 후 다시 실행
  2. 기기 재시작: 기기를 재부팅
  3. 캐시 정리: 앱 설정에서 캐시 삭제
  4. 업데이트 확인: 최신 버전으로 업데이트
  5. 저장공간 확인: 기기 저장공간이 충분한지 확인

문제가 지속되면 고객지원팀에 문의해주세요.

알림이 오지 않아요

알림 문제 해결 방법:

  • 앱 알림 설정: 기기 설정 > 앱 > Goal-Illa > 알림 허용
  • 방해금지 모드: 방해금지 모드가 활성화되어 있는지 확인
  • 앱 내 설정: Goal-Illa 앱 설정에서 알림 옵션 확인
  • 배터리 최적화: 배터리 최적화에서 앱 제외

원하는 답변을 찾지 못하셨나요?

고객지원팀에 직접 문의해주세요. 빠르고 정확한 답변을 드리겠습니다.
